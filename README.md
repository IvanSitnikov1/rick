#### Описание  
Приложение на React, работающее с API по сериалу Рик и Морти. Позволяет посматривать персонажей по отдельным эпизодам и локациям. Так же показывает цветом состояние персонажа(жив, мертв, не определено)

#### Запуск приложени:  
npm start
